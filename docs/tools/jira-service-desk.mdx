### Get Service Desk For Project

Get the Jira Service Desk associated with a project key.

**Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `project_key` | `string` | Yes | Jira project key (e.g., 'SUP') |

---

### Get Service Desk Queues

Get queues for a Jira Service Desk.

**Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `service_desk_id` | `string` | Yes | Service desk ID (e.g., '4') |
| `start_at` | `integer` | No | Starting index for pagination (0-based) |
| `limit` | `integer` | No | Maximum number of results (1-50) |

### Get Customer Request

### Search Customer Requests

Search Jira Service Management customer requests visible to the caller.

**Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `request_ownership` | `string` | No | Request ownership filter |
| `request_status` | `string` | No | Optional request status filter |
| `search_term` | `string` | No | Optional search text |
| `organization_id` | `string` | No | Optional organization ID filter |
| `service_desk_id` | `string` | No | Optional service desk ID filter |
| `start_at` | `integer` | No | Starting index for pagination (0-based) |
| `limit` | `integer` | No | Maximum number of results (1-50) |
| `expand` | `string` | No | Optional comma-separated expand values |
