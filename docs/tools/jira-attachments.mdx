title: "Jira Attachments"
description: "Download attachments and render attached or editor-embedded images"
---

### Download Attachments

Download attachments from a Jira issue.

**Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `issue_key` | `string` | Yes | Jira issue key (e.g., 'PROJ-123', 'ACV2-642') |
**Example:**

```json
{"issue_key": "PROJ-123"}
```

<Tip>
Downloads all attachments from the issue. Files larger than 50MB are skipped. Returns base64-encoded content.
</Tip>


---

### Get Issue Images

Get all images attached to a Jira issue as inline image content.

**Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `issue_key` | `string` | Yes | Jira issue key (e.g., 'PROJ-123'). Returns image attachments as inline ImageContent for LLM vision. |

---

### Get Issue Embedded Images

Get images pasted directly into a Jira issue's description or comments as inline image content.

**Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `issue_key` | `string` | Yes | Jira issue key (e.g., 'PROJ-123'). Returns editor-embedded images that are not exposed as formal attachments. |

<Tip>
Use this together with `jira_get_issue_images` when an issue may contain both formal image attachments and images pasted into rich-text fields.
</Tip>
